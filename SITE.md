# Trendwire - Site Vision & Roadmap

## Vision
Trendwire is a premium, real-time intelligence engine for AI enthusiasts and digital marketers. It is not a blog; it is a "Growth Engine" that aggregates high-signal updates, tools, and reports.

## Architecture
- **Type**: Static Web App (HTML/CSS/JS)
- **Styling**: Tailwind CSS (CDN)
- **Icons**: Lucide (CDN)
- **Font**: Inter (Body), JetBrains Mono (Headers/Data)

## Roadmap
- [ ] **v1.0**: Core Feed, Paywall (5 article limit), Basic "Auth" (Mock), Mobile Responsive.
- [ ] **v1.1**: User Accounts (Supabase), Real Payment Integration (Stripe).
- [ ] **v1.2**: Personalized Feeds, "Deep Dive" Reports.

## Sitemap
- `index.html`: Main Dashboard (Feed, Sidebar, Ticker).
